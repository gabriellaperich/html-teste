Tags

Algumas Tags principais utilizadas em HTML:

Aqui será demonstrado alguns códigos mais utilizados na linguagem HTML. É sempre bom ressaltar que o HTML NÃO é uma linguagem de programação, como por exemplo Java, Phyton, Ruby, PHP e etc. E o HTML é uma linguagem de marcação utilizada para o desenvolvimento front-end, ou seja, diretamente ligado ao design do site, a interface de usuário e o cliente. Ou seja, o externo do site. Sempre é importante lembrar também que o HTML traduzido se refere a Linguagem de Marcação de Hiper-Texto.

Tags:

< html > - se refere a uma tag introdutória que é essencial para a formatação do site e seu desenvolvimento.
< head > - refere-se a parte que é carregada previamente do site.
< body > - é o corpo do site, literalmente. É nele que estão contidas as informações que o cliente vê assim que o site é carregado.

Estas três tags são a base da estrutura HTML, apesar de ser possível adicionar tags adicionais para melhorar a estrutura do site.
